API

Das System stellt eine REST-API zur Verfügung, die für Integrationen mit externen Anwendungen genutzt werden kann. Die API erlaubt das automatisierte Auslesen und Übertragen von Stammdaten, Benutzerinformationen und weiteren Strukturdaten des Systems.

Swagger Dokumentation

Die API ist vollständig über eine integrierte Swagger Oberfläche dokumentiert. Diese dient gleichzeitig als Referenz, Testwerkzeug und interaktive Beschreibung der vorhandenen Endpunkte.
